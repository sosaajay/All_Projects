Sales & Customer Analytics Dashboard (Power BI)
Project Overview

This Power BI project transforms raw transactional data into meaningful business insights using an interactive and visually rich dashboard. It focuses on analyzing sales performance, customer behavior, product trends, and return patterns.

The report is built using a star schema data model and provides decision-makers with clear, actionable insights through dynamic visuals and KPIs.

Objectives
Track overall sales performance
Analyze customer segments
Identify top-performing products
Monitor returns and order trends
Visualize regional sales distribution
Support data-driven decision making
