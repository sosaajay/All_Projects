
# Analyzing Income Distribution and Household Demographics

## Project Overview

This project analyzes household demographic and income data using Python.
The objective is to perform descriptive statistical analysis, understand income distribution patterns, and visualize relationships between demographic variables.

The project includes:
- Data type identification
- Measures of central tendency
- Measures of dispersion
- Distribution analysis
- Skewness and kurtosis calculation
- Data visualization using graphs

---

## Dataset Information

The dataset contains 200 household records with the following columns:

| Column Name | Data Type | Description |
|---|---|---|
| Household_ID | Categorical | Unique household identifier |
| Age_of_Household_Head | Numerical | Age of household head |
| Household_Income | Numerical | Monthly household income |
| Education_Level | Categorical | Education level |
| Family_Size | Numerical | Number of family members |
| Owns_House | Categorical | Yes or No |
| Urban_Rural | Categorical | Urban or Rural |

## Project Tasks

### 1. Types of Data
Identify categorical and numerical columns.

### 2. Central Tendency
Calculate:
- Mean
- Median
- Mode

For:
- Household Income
- Age of Household Head

### 3. Measures of Dispersion
Calculate:
- Range
- Variance
- Standard Deviation
- Interquartile Range (IQR)

### 4. Distribution Analysis
Perform:
- Histogram plotting
- Gaussian distribution fitting
- Skewness calculation
- Kurtosis calculation

### 5. Visualizations
Generate:
- Histogram with KDE
- Boxplot for income vs education level
- Scatter plot for age vs income
