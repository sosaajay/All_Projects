# 📚 Statistical Theory Overview 🧠

## 📊 Statistical Distribution

A statistical distribution describes how values of a random variable are spread and how frequently they occur.

Important parameters:

📌 Mean → Average value

📌 Variance → Data spread

📌 Skewness → Data asymmetry

📌 Kurtosis → Tail behaviour

---

## 📈 Q-Q Plot

Q-Q Plot (Quantile-Quantile Plot) compares sample data with a theoretical distribution.

Used for:

✅ Checking distribution behaviour

✅ Detecting skewness

✅ Identifying extreme values

---

## 🔢 Discrete and Continuous Distribution

### Discrete Distribution

Used for countable values.

Example:

🔢 Number of transactions per day


### Continuous Distribution

Used for measurable values.

Example:

💰 Transaction amount

---

## 🎯 Bernoulli Distribution

Represents two possible outcomes:

✅ Success = 1

❌ Failure = 0


Project use:

Calculates transaction success probability.

---

## 🎲 Binomial Distribution

Represents number of successes from multiple Bernoulli trials.

Project use:

Simulates successful transactions out of multiple attempts.

---

## 💰 Log-Normal Distribution

Used for positive and right-skewed data.

Project use:

Analyses transaction amounts because values are positive and may contain high-value transactions.

---

## ⚡ Power Law Distribution

Used to identify heavy-tail behaviour.

Project use:

Finds extreme high-value transaction patterns.

---

## 🔄 Box-Cox Transformation

Used to improve data distribution.

Benefits:

✅ Reduces skewness

✅ Improves normality

---

## ⏱️ Poisson Distribution

Used for counting events in a fixed time period.

Project use:

Analyses daily transaction counts.

---

## 📐 Z-Score Probability

Measures how far a value is from the mean.

Used for:

🚨 Outlier detection

📊 Probability calculation

---

## 📊 PDF and CDF

PDF:

Shows probability density of continuous data.


CDF:

Shows cumulative probability up to a value.
