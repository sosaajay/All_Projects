# 📊 Statistical Analysis Using Python

## 📌 Project Overview

This project demonstrates the practical implementation of statistical concepts using Python. A student dataset is analyzed with the help of various statistical techniques including measures of central tendency, measures of dispersion, probability, probability distributions, data visualization, and basic linear algebra operations.

The project uses Python libraries such as NumPy, Pandas, Matplotlib, Seaborn, and SciPy to perform calculations and generate visual insights from the dataset.

---

## 🎯 Objectives

- Perform statistical analysis on student performance data.
- Calculate measures of central tendency and dispersion.
- Apply probability concepts on real-world data.
- Analyze data distribution using statistical plots.
- Perform basic linear algebra operations.
- Visualize data using graphs and statistical charts.

---

## 🛠 Technologies and Libraries Used

- NumPy
- Pandas
- Matplotlib
- Seaborn
- SciPy

---

## 📂 Dataset

The project uses **Student_Data.csv** containing:
- Math Score
- Science Score
- English Score
- Hours Studied
- Pass/Fail Status

---

## 🚀 Project Implementation

### 1. Dataset Loading and Exploration
- Dataset imported using Pandas
- Used head(), tail(), info(), describe()
- Checked missing values and dataset structure

### 2. Measures of Central Tendency and Dispersion
- Mean, Median, Mode of Math Score
- Range, Variance, Standard Deviation of Science Score

### 3. Probability Analysis
- Probability of passing
- Contingency table analysis
- Conditional probability P(Pass | Hours_Studied > 5)

### 4. Distribution Analysis and Visualization
- Histogram of Math Score
- Skewness and Kurtosis of Science Score
- Q-Q Plot of English Score

### 5. Linear Algebra Operations
- Vector representation
- Dot Product
- L1 Norm
- L2 Norm
- Angle Between Vectors

---

## 📈 Insights

- Average Math Score indicates overall performance.
- Students studying more than 5 hours have a higher chance of passing.
- Math Scores are approximately normally distributed.
- Skewness and Kurtosis describe the shape of score distributions.
- Linear Algebra demonstrates vector-based analysis of data.

This project successfully applies statistical analysis techniques using Python and provides practical experience with data science concepts including descriptive statistics, probability, visualization, and linear algebra.
