# Probability & Statistics Assignment

## Project Description
This project is based on probability concepts using a student dataset.  
The objective of the assignment is to apply statistical techniques and probability formulas on real dataset values.

---

# Dataset Information

The dataset contains the following fields:

| Field | Description |
|-------|-------------|
| study_hours | Number of hours a student studied per week |
| attendance | Percentage attendance in lectures |
| group_discussion | Participation in group discussions (Yes/No) |
| previous_test_score | Marks out of 100 from previous test |
| final_exam_pass | Whether the student passed the final exam (Pass/Fail) |
