STUDENT PERFORMANCE & ATTENDANCE TRACKER
DBMS PROJECT

Project Name:
Student Performance & Attendance Tracker

Objective:
This project is used to manage student details, courses, attendance and marks using MySQL database.
It helps to easily track student performance and attendance.

Tools Used:
Database – MySQL
Language – SQL
Platform – Windows

Database Name:
sttracker

Tables Used:
department – department details
student – student info
faculty – faculty info
course – course details
enrollment – student course mapping
attendance – attendance record
grade – student marks

Project Features:
• manage student and department data
• assign faculty to courses
• enroll students in courses
• mark student attendance
• store student marks
• calculate attendance percentage
• show result based on marks

SQL Concepts Used:
create, insert, select
update, delete
joins
group by, having
subquery
view
case statement

Result Logic:
marks >= 85 → Distinction
marks >= 70 → First Class
marks >= 50 → Pass
marks < 50 → Fail

Conclusion:
This DBMS project shows how student performance and attendance can be managed in an easy and effective way using SQL.
