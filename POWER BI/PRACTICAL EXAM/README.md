# 🎓 Student Performance Dashboard using Power BI

## 📌 Introduction
This project presents an interactive **Student Performance Dashboard** built using Power BI to analyze academic performance, attendance, and behavioral patterns of students.

The dashboard includes multiple views such as a summary dashboard and a detailed student-level drillthrough page. It also supports mobile layout for better accessibility.

---

## 🎯 Objective
- Analyze overall student performance across subjects  
- Track monthly attendance trends  
- Evaluate student behavior distribution  
- Provide detailed insights at an individual student level  
- Enable interactive filtering and dynamic analysis  

---

## 📈 Key Highlights
- Total Students: 1000  
- Overall Attendance: ~90%  
- Average Score: ~50  
- Average Percentage Score: ~50%  
- Total Days Recorded: 100000  
- Present Days: 90K  
- Absent Days: 10K

## 🧩 Dashboard Structure

### 🔹 Main Dashboard View

---

### 🔹 Student Drillthrough View

## 🔄 Navigation & Features

### 🔖 Bookmarks
- Used to switch between:
  - Academic View  
  - Behavior View  
  - Default View  

### 🔍 Drillthrough
- Navigate from main dashboard to detailed student-level page  
- Enables deep analysis of individual student performance
