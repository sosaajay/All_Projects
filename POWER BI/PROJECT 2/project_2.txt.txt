Power BI Project Instructions 

Step 1: Import Data
I opened Power BI Desktop and used Get Data → Excel.
Then I imported these files:
CustomerDim.xlsx, DateDim.xlsx, ProductDim.xlsx, RegionDim.xlsx, ReturnsFact.xlsx, SalesFact.xlsx
After that, I clicked on Transform Data.

Step 2: Data Cleaning
In Power Query, I removed blank rows and fixed data types:
IDs → Whole Number, Quantity → Whole Number, Revenue → Currency,
Discount → Decimal, Date → Date, Month → Text, Year → Whole Number.
Then I clicked Close & Apply.

Step 3: Relationships
In Model View, I created relationships:
SalesFact → CustomerDim, ProductDim, RegionDim, DateDim
ReturnsFact → SalesFact
ReturnsFact → DateDim (Inactive)
Cross filter direction was set to Single.

Step 4: Star Schema
SalesFact is the main table and all other tables are connected to it.
This structure is called Star Schema.

Step 5: Cardinality
1 : Many → CustomerDim → SalesFact
Many : 1 → SalesFact → ProductDim
1 : 1 → SalesFact → ReturnsFact

Step 6: Settings
Cross filter = Single
Bi-directional filter not used.

Step 7: Formatting
Revenue → Currency
Quantity → Whole Number
Discount → Percentage
Date → Date

Step 8: Categories
City → City, State → State, Country → Country, ProductName → Product

Step 9: Hierarchies
Date: Year → Quarter → Month → Date
Region: Country → State → City
Product: Category → Subcategory → ProductName

Step 10: Verification (Matrix)
1. Sales by Category & Region
2. Return Reasons by Fiscal Year (used DAX for inactive relationship)
3. Revenue by Customer Segment

Final Result
Data cleaned, relationships created, star schema applied,
hierarchies built, and results verified using matrix visuals.
