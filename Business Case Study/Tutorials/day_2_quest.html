Measures of Central Tendency — Mean, Median, Mode
+30 XP on scroll
Central Tendency kya hota hai?
Central Tendency = ek single value jo poore dataset ko represent kare — "data ka center" dhundho. Teen main measures hain: Mean (arithmetic average), Median (middle value), Mode (most frequent value).
Mean — Arithmetic Average
POPULATION MEAN & SAMPLE MEAN
μ = (Σ xᵢ) / N     (Population)      x̄ = (Σ xᵢ) / n     (Sample)
μ / x̄ = Mean Σ xᵢ = Sum of all values N / n = Total count
Worked Example — IPL Runs
Data: [45, 82, 0, 61, 34, 78, 12, 95, 56, 37]
Sum = 45+82+0+61+34+78+12+95+56+37 = 500
n = 10 matches
Mean (x̄) = 500 / 10 = 50.0 runs
Mean ki Weakness
Mean outliers se bahut sensitive hai!
Agar ek match mein 200 runs (impossible lekin) → Mean bahut badh jaayega, poori picture distort ho jaayegi.
Income data mein Ambanis ka salary average distort karta hai!

Median — Middle Value
MEDIAN FORMULA
Odd n: Median = value at position [(n+1)/2]
Even n: Median = average of values at [n/2] and [n/2 + 1]
Step 1: Sort data ascending. Step 2: Find middle position.
IPL Example — Median Calculate Karo
Sorted: [0, 12, 34, 37, 45, 56, 61, 78, 82, 95]  ← n=10 (even)
Position n/2 = 5th value = 45, position 6th = 56
Median = (45 + 56) / 2 = 50.5 runs
Median ki Strength
Mean se better jab outliers ya skewed data ho!
Income data pe — median income more representative hai average income se. 50% log median ke upar, 50% neeche!
RBI "median household income" report karta hai — not mean!

Mode — Most Frequent Value
MODE DEFINITION
Mode = value(s) that appear(s) most frequently in the dataset
Unimodal (1 mode) · Bimodal (2 modes) · Multimodal (3+ modes) · No mode (all unique)
Mode Examples — India
Shoe sizes at Bata store: [6, 7, 8, 8, 7, 9, 8, 6, 8, 10] → Mode = 8 (appears 4 times!)

Zomato most ordered dish in Delhi: [Biryani, Butter Chicken, Biryani, Pizza, Biryani] → Mode = Biryani

Mode works for both numerical AND categorical data — Mean/Median do not!
Kab Kaunsa Use Karein?
Mean → Normal distribution, no outliers. Exam marks, temperature

Median → Skewed data, outliers present. Income, house prices

Mode → Categorical data, most popular item. Election results, product sizes

Skewed income data: Mean > Median (right skew). Report Median!

Measures of Dispersion — Range, Variance, Standard Deviation

Dispersion kyun zaroori hai?
Do datasets ka mean same ho sakta hai lekin spread bilkul alag! Dispersion measures batate hain — data mean ke aaspaas kitna spread hai. Same mean, very different spread = very different datasets!

Example: Cricketer A = [48, 50, 52, 49, 51] | Cricketer B = [10, 90, 5, 80, 65] → Both mean ≈ 50, lekin B bahut inconsistent hai!

Range — Sabse Simple Dispersion Measure
RANGE FORMULA
Range = Maximum Value − Minimum Value
IPL Example: Max=95, Min=0 → Range = 95 − 0 = 95 runs
Range ke Advantages
✅ Bahut simple calculate karna
✅ Quick idea of spread
✅ Easy to understand
🇮🇳 IMD weather: "Temperature range: 12°C to 40°C = 28°C range today"
Range ki Weakness
❌ Sirf 2 extreme values use karta hai — baaki ignore!
❌ Outliers se bahut affected
❌ Data ka real spread nahi batata
Ek outlier se Range dramatically change ho sakta hai!
Variance — Average Squared Deviation
VARIANCE FORMULAS
Population: σ² = Σ(xᵢ − μ)² / N
Sample: s² = Σ(xᵢ − x̄)² / (n − 1)
Note: Sample formula mein (n−1) use hota hai (Bessel's correction) — unbiased estimate ke liye!
Mean calculate karo → x̄ = 50.0 (IPL data)
Pehle mean dhundho — ye reference point hai.
Har value ka deviation find karo → (xᵢ − x̄)
45−50=−5, 82−50=+32, 0−50=−50, 61−50=+11, 34−50=−16,
78−50=+28, 12−50=−38, 95−50=+45, 56−50=+6, 37−50=−13
Deviations square karo → (xᵢ − x̄)²
25, 1024, 2500, 121, 256, 784, 1444, 2025, 36, 169
Sum of squares = 8384
Average karo → s² = 8384 / (10−1)
s² = 8384 / 9 = 931.6 runs²
Unit = runs² — not directly interpretable! Isliye Standard Deviation use karte hain.
Variance mein Square Kyun?
✅ Cancellation rokne ke liye: Negative deviations positive se cancel ho jaate → Square se sab positive!
✅ Large deviations ko zyada penalize: 10 unit deviation → 100, but 2 unit → 4 (not 5×)
❌ Problem: Units squared ho jaate hain → runs² meaningless!
→ Solution: Square root lete hain → Standard Deviation original unit mein!
Standard Deviation — Variance ka Square Root
STANDARD DEVIATION FORMULAS
Population: σ = √[Σ(xᵢ − μ)² / N]
Sample: s = √[Σ(xᵢ − x̄)² / (n − 1)]
IPL Example: s = √931.6 = 30.52 runs → Original unit mein! Interpretable!
Std Dev ka Matlab
s = 30.52 runs → Average mein Virat ki runs mean (50) se ~30.52 runs dur hoti hain.

High Std Dev = inconsistent
Low Std Dev = consistent
Empirical Rule (68-95-99.7)
Normal distribution mein:
→ 68% data: μ ± 1σ ke andar
→ 95% data: μ ± 2σ ke andar
→ 99.7% data: μ ± 3σ ke andar
3σ se bahar = rare outlier!
India Examples
🏏 Virat (s=30.5): Somewhat inconsistent
🌡️ Mumbai temp: Low std dev → mild climate year-round
💰 Stock returns: High std dev → volatile/risky
📊 SEBI volatility: Std dev of daily returns = risk measure

Final Message
Descriptive Statistics = jo data hai usse describe karo — predict nahi karo, just summarize!
→ Mean = arithmetic average, outliers distort karte hain. Median = middle value, outlier-proof. Mode = most frequent, categorical data ke liye!
→ Range = simple lekin sirf extremes use karta hai. Variance = squared deviations, ML mein use. Std Dev = √Variance, interpretable, risk measure!
→ Golden Rule: Central tendency + Dispersion dono mil ke data ka poora story batate hain — ek without doosre incomplete picture hai!
